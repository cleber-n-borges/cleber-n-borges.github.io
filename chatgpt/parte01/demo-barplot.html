Vou criar vários exemplos usando a função barplot do R, demonstrando diferentes usos e personalizações. A barplot é utilizada para criar gráficos de barras e oferece uma grande variedade de opções para personalização.

1. Exemplo Básico de barplot

r
# Dados de exemplo
valores <- c(5, 10, 15, 20)

# Criação de um gráfico de barras básico
barplot(valores, 
        main = "Gráfico de Barras Simples", 
        xlab = "Categorias", 
        ylab = "Valores", 
        col = "skyblue")

Explicação:

  • valores: Um vetor numérico com os dados a serem plotados.
  • barplot(valores, ...): Cria o gráfico de barras.
  • main: Título principal do gráfico.
  • xlab e ylab: Rótulos dos eixos X e Y.
  • col: Cor das barras.

2. Gráfico de Barras com Várias Categorias

r
# Dados de exemplo
matriz_valores <- matrix(c(5, 10, 15, 20, 3, 6, 9, 12), nrow = 2)

# Criação de gráfico de barras com múltiplas categorias
barplot(matriz_valores, 
        beside = TRUE, 
        main = "Gráfico de Barras com Múltiplas Categorias", 
        xlab = "Categorias", 
        ylab = "Valores", 
        col = c("red", "blue"),
        legend.text = c("Grupo 1", "Grupo 2"))

Explicação:

  • matriz_valores: Uma matriz onde cada linha representa uma série de dados (categorias) para o gráfico.
  • beside = TRUE: Coloca as barras lado a lado (ao invés de empilhadas).
  • col: Define cores diferentes para cada grupo de barras.
  • legend.text: Adiciona uma legenda para identificar cada grupo de barras.

3. Gráfico de Barras Empilhadas

r
# Dados de exemplo
matriz_valores <- matrix(c(5, 10, 15, 20, 3, 6, 9, 12), nrow = 2)

# Criação de gráfico de barras empilhadas
barplot(matriz_valores, 
        beside = FALSE, 
        main = "Gráfico de Barras Empilhadas", 
        xlab = "Categorias", 
        ylab = "Valores", 
        col = c("green", "purple"),
        legend.text = c("Grupo 1", "Grupo 2"))

Explicação:

  • beside = FALSE: Empilha as barras umas sobre as outras.
  • As outras opções são similares ao exemplo anterior.

4. Gráfico de Barras com Barras Horizontais

r
# Dados de exemplo
valores <- c(5, 10, 15, 20)

# Criação de gráfico de barras horizontal
barplot(valores, 
        horiz = TRUE, 
        main = "Gráfico de Barras Horizontal", 
        xlab = "Valores", 
        ylab = "Categorias", 
        col = "orange")

Explicação:

  • horiz = TRUE: Gera um gráfico de barras horizontal.
  • xlab e ylab foram invertidos para refletir a mudança de orientação.

5. Gráfico de Barras com Personalização de Eixos e Rótulos

r
# Dados de exemplo
valores <- c(5, 10, 15, 20)

# Criação de gráfico de barras com personalização de eixos e rótulos
barplot(valores, 
        names.arg = c("A", "B", "C", "D"), 
        main = "Gráfico de Barras com Rótulos Personalizados", 
        xlab = "Categorias", 
        ylab = "Valores", 
        col = "cyan", 
        las = 1)

Explicação:

  • names.arg: Define rótulos personalizados para cada barra.
  • las = 1: Gira os rótulos do eixo Y para ficarem horizontais.

6. Gráfico de Barras com Erros (Bar Error)

r
# Dados de exemplo
valores <- c(5, 10, 15, 20)
erro <- c(1, 2, 1.5, 1)

# Criação de gráfico de barras com barras de erro
bar <- barplot(valores, 
               ylim = c(0, 25), 
               main = "Gráfico de Barras com Erros", 
               xlab = "Categorias", 
               ylab = "Valores", 
               col = "lightblue")

# Adicionando barras de erro
arrows(bar, valores + erro, bar, valores - erro, 
       angle = 90, 
       code = 3, 
       length = 0.1)

Explicação:

  • ylim: Ajusta o limite do eixo Y para acomodar as barras de erro.
  • arrows: Adiciona barras de erro no gráfico, utilizando as coordenadas das barras (bar), valores superiores e inferiores (definidos pelos erros).

7. Gráfico de Barras com Anotação de Texto

r
# Dados de exemplo
valores <- c(5, 10, 15, 20)

# Criação de gráfico de barras
bar <- barplot(valores, 
               main = "Gráfico de Barras com Anotação de Texto", 
               xlab = "Categorias", 
               ylab = "Valores", 
               col = "pink")

# Adicionando texto em cada barra
text(bar, valores, labels = valores, pos = 3, cex = 0.8, col = "black")

Explicação:

  • text: Adiciona texto diretamente sobre as barras, com o valor de cada barra.
  • pos = 3: Coloca o texto acima das barras.
  • cex: Controla o tamanho do texto.

Esses exemplos cobrem uma variedade de usos da função barplot, desde gráficos básicos até personalizações avançadas.
